【run】の意味・使い方・例文 ― operateとの違い【TOEIC英単語】

🔰 runの発音と意味

[動詞] 走る、運営する、作動する

run は、「走る」という基本的な意味のほか、「会社やお店を運営する」「機械などを動かす」といった幅広い意味で使われます。

語源は古英語の「rinnan（走る）」に由来し、そこから「動かす」「管理する」といった意味に広がりました。

他の品詞：

  • runner：走者（名詞）
  • running：走ること、運営（名詞・形容詞）

✏️ runの例文

【日常会話】
I run every morning in the park.
　私は毎朝公園で走ります。

【ビジネス英語】
She runs a small coffee shop downtown.
　彼女は街の中心部で小さなカフェを経営しています。

🎯 TOEICの出題頻度と傾向

  • 難易度：★★☆☆☆
  • 出題パート：Part 3（会話問題）

runはTOEICの会話パートで「運営する」「作動する」などの意味でよく出題されます。日常的な動作やビジネスシーンの両方で頻出です。

❓ クイズに挑戦！

自然な使い方はどっち？

  1. The manager runs the store efficiently.
  2. The manager runs the store efficiency.

✅ クイズの答えと解説

  • 正解：1
  • 解説：efficiently（効率的に）は副詞でrunを修飾します。efficiency（効率）は名詞なので文法的に不自然です。

🤔 operate との違いは？

  • run：運営や管理、または走る・作動するなど幅広い意味で使う
  • operate：機械やシステムを操作・運転する意味が中心

runは人や組織が「全体を動かす・管理する」ニュアンスが強く、operateは「機械や装置を直接操作する」場面で使います。

🧩 あわせて覚えたい

  • manage：管理する、運営する
  • control：制御する、管理する

📖 できればこれも

  • move：動く、動かす
  • lead：導く、率いる
