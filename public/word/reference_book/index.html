【reference book】の意味・使い方・例文 ― textbookとの違い【TOEIC英単語】

🔰 reference bookの発音と意味

[名詞] 参考書（可算）

特定の情報や知識を調べるために使う本のことを指します。辞書や百科事典、年鑑などがこれにあたります。学習や調査の際に必要な情報を探す目的で利用されます。

referenceはラテン語のreferre（持ち帰る、報告する）に由来し、「参照」や「参考」を意味します。bookは古英語のbōc（本）から来ており、referenceとbookを組み合わせた複合語です。

✏️ reference bookの例文

【日常会話】
You can find the answer in a reference book.
　参考書で答えを見つけることができます。

She borrowed a reference book from the library.
　彼女は図書館から参考書を借りました。

【ビジネス英語】
When writing your report, please use at least one reference book to support your arguments.
　レポートを書く際は、主張を裏付けるために少なくとも1冊の参考書を使ってください。

🤔 textbook との違いは？

  • reference book：必要な情報を調べるための本というニュアンスがあります。
  • textbook：学習内容を体系的に教える本というニュアンスがあります。

reference bookは情報を調べるために使う本で、textbookは学習内容を順序立てて学ぶための本です。reference bookは調査や確認、textbookは授業や独学に適しています。
