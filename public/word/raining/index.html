【raining】の意味・使い方・例文 ― snowingとの違い【TOEIC英単語】

🔰 rainingの発音と意味

[動詞] 雨が降っている

空から雨が降っている状態を表す表現です。天気について話すときによく使われます。

この単語は古英語の“regnian”（雨が降る）に由来し、動詞“rain”に現在進行形を表す“ing”が付いた形です。

✏️ rainingの例文

【日常会話】
It is raining outside.
　外は雨が降っています。

I forgot my umbrella because it wasn’t raining this morning.
　今朝は雨が降っていなかったので傘を忘れました。

【ビジネス英語】
Due to the fact that it is raining heavily, the outdoor event has been postponed until next week.
　激しい雨が降っているため、屋外イベントは来週まで延期されました。

🎯 TOEICの出題頻度と傾向

  • 難易度：★☆☆☆☆
  • 出題パート：Part1（写真描写問題）

天気や屋外の描写を問う写真描写問題（Part1）で頻出です。進行形の形でよく出題されます。

❓ クイズに挑戦！

正しい使い方はどっち？

  1. We canceled the picnic because it was raining.
  2. We canceled the picnic because it was snowing.

✅ クイズの答えと解説

  • 正解：1
  • 解説：rainingは「雨が降っている」という意味です。2番は「雪が降っている」となり、意味が異なります。

TOEICテスト対策

🤔 snowing との違いは？

  • raining：雨が降っているという事実を淡々と伝える表現です。
  • snowing：雪が降っている状態を表し、寒さや冬のイメージが強い表現です。

rainingは雨が降る状況、snowingは雪が降る状況を表し、どちらも天気の進行中の状態を示しますが、降っているものが異なります。

🧩 あわせて覚えたい

  • pouring（土砂降り）
  • drizzling（小雨）

📖 できればこれも

  • cloudy（くもり）
  • windy（風が強い）
