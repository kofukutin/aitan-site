【register】の意味・使い方・例文 ― enrollとの違い【TOEIC英単語】

🔰 registerの発音と意味

[動詞] 登録する、記録する

register は、「公式に名前や情報を登録する」「記録する」という意味で、ビジネスや公式な場面でよく使われます。

語源はラテン語の「regesta（記録）」から来ており、「リストに載せる」というイメージです。

他の品詞：

  • registration：登録（名詞）
  • registrant：登録者（名詞）

✏️ registerの例文

【日常会話】
You need to register before using the app.
　アプリを使う前に登録が必要です。

【ビジネス英語】
Please register for the conference by Friday.
　金曜日までに会議への登録をお願いします。

🎯 TOEICの出題頻度と傾向

  • 難易度：★★☆☆☆
  • 出題パート：Part 5（文法・語彙問題）

TOEICでは「登録する」「記録する」という意味で、申込や手続きに関する文脈でよく出題されます。名詞形（registration）も頻出です。

❓ クイズに挑戦！

正しい使い方はどっち？

  1. You need to register for the event online.
  2. You need to register the event for online.

✅ クイズの答えと解説

  • 正解：1
  • 解説：「register for ～」で「～に登録する」となり、正しい使い方です。

🤔 enroll との違いは？

  • register：公式なリストやシステムに登録すること全般を指す
  • enroll：特に学校や講座などに「入学・入会」する意味が強い

registerは幅広い登録に使えますが、enrollは教育や講座など「参加・入学」のニュアンスが強いです。

  • record：記録する、記載する
