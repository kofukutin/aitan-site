【able】の意味・使い方・例文 ― capableとの違い【TOEIC英単語】

🔰 ableの発音と意味

[形容詞] ～できる、能力がある

able は、「～することができる」「能力がある」という意味で、主に be able to ～ の形で使われます。
日常会話からビジネスまで幅広く使われ、何かを実行する力や可能性があることを表します。

語源はラテン語の habilis（適した、能力がある）に由来し、英語では「できる」という意味で定着しました。

他の品詞：

  • ability：能力（名詞）
  • ably：巧みに（副詞）

✏️ ableの例文

【日常会話】
I am able to swim.
　私は泳ぐことができます。

【ビジネス英語】
She is able to handle multiple tasks efficiently.
　彼女は複数の業務を効率的にこなすことができます。

🎯 TOEICの出題頻度と傾向

  • 難易度：★★☆☆☆
  • 出題パート：Part 5（文法・語彙問題）

TOEICでは「be able to ～」の形で、canとの使い分けや時制表現でよく出題されます。
特に文法問題で頻出です。

❓ クイズに挑戦！

正しい使い方はどっち？

  1. I am able to finish the report by tomorrow.
  2. I am able finish the report by tomorrow.

✅ クイズの答えと解説

  • 正解：1
  • 解説：「be able to」の後は必ず動詞の原形が続くため、toが必要です。

🤔 capable との違いは？

  • able：状況や条件が整えば「できる」ことを表す、一般的な表現。
  • capable：訓練や経験により「有能である」「能力が高い」ことを強調。

「able」は単に可能性や状況を指し、「capable」はより高い能力や専門性を強調します。

🧩 あわせて覚えたい

  • unable：できない
  • qualified：資格がある、適任の

📖 できればこれも

  • available：利用できる、手が空いている
  • efficient：効率的な
