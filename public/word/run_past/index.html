【run past】の意味・使い方・例文 ― consultとの違い【TOEIC英単語】

🔰 run pastの発音と意味

[熟語] （人）に（案や考えなど）をざっと伝えて意見を求める

自分の考えや計画などを相手に簡単に伝えて、意見や反応を求めるときに使います。特にビジネスや会議の場面で、正式な相談よりも気軽に確認したいときによく使われます。

「run」は「走る」、「past」は「〜のそばを通り過ぎる」という意味を持つ英単語です。これらが組み合わさり、「何かを（人の前を通すように）ざっと見せる、伝える」という比喩的な意味で使われるようになりました。

✏️ run pastの例文

【日常英会話】
Can I run this idea past you?
　このアイデアについてあなたの意見を聞いてもいいですか？

Let me run my plan past the team first.
　まずチームに私の計画を伝えてみます。

【ビジネス英語】
Before we make a final decision, I’d like to run the proposal past our manager to get her feedback.
　最終決定をする前に、この提案についてマネージャーの意見を聞きたいです。

🎯 TOEICの出題頻度と傾向

  • 難易度：★★★☆☆
  • 出題パート：Part3（会話問題）

会議や相談の場面で、意見を求める表現として頻出します。Part3の会話問題でよく見かけます。

❓ クイズに挑戦！

正しい使い方はどっち？

  1. He ran past the finish line quickly.
  2. She wants to run her new idea past her boss.

✅ クイズの答えと解説

  • 正解：2
  • 解説：1は「走って通り過ぎる」という物理的な意味になり、熟語の意味と異なります。2は「アイデアを上司に伝えて意見を求める」という意味で正しいです。

🤔 consult との違いは？

  • run past：気軽に意見や反応を求める柔らかい印象です。
  • consult：専門的な助言や正式な相談を求める、やや堅い印象です。

run pastはカジュアルに意見を聞くときに使い、consultは専門的・正式な相談や助言を求める場面で使います。
