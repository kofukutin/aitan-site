🔰 slashmarkの発音と意味

[名詞] 斜線記号（可算）

文章やコンピュータ上で使われる斜めの線「/」を指します。区切りや選択肢の表示、ファイルパスの区切りなど、さまざまな場面で使われます。

slashmarkは「slash（斜線）」と「mark（記号）」が組み合わさった合成語です。英語のslashはラテン語の「slasciare（切る）」に由来し、markはゲルマン語系の「marka（印）」に由来しています。

✏️ slashmarkの例文

【日常英会話】
Please write your answer with a slashmark.
　スラッシュマークを使って答えを書いてください。

The website address uses a slashmark.
　ウェブサイトのアドレスにはスラッシュマークが使われています。

【ビジネス英語】
In the file path, each folder is separated by a slashmark to indicate the directory structure.
　ファイルパスでは、各フォルダがスラッシュマークで区切られてディレクトリ構造を示します。

自然な使い方はどっち？

  1. He wrote his phone number with a hyphenmark.

🤔 hyphen との違いは？

  • slashmark：記号としての斜線を指し、区切りや選択肢など幅広い用途で使われます。
  • hyphen：単語や数字をつなぐ短い横線を指し、主に語句の連結や範囲の表現に使われます。

slashmarkは斜めの線で区切る場合に使い、hyphenは横線で単語や数字をつなぐ場合に使います。用途や見た目が異なります。
