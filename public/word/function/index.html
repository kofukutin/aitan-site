【function】の意味・使い方・例文 ― roleとの違い【TOEIC英単語】

🔰 functionの発音と意味

[名詞] 機能、役割、関数（可算）

function は、「物や人が果たす役割」や「機能」、またITや数学分野では「関数」という意味で使われます。

語源はラテン語の「fungi（果たす）」から来ており、「何かを果たすこと」が基本イメージです。

他の品詞：

  • functional：機能的な（形容詞）
  • functionally：機能的に（副詞）

✏️ functionの例文

【日常会話】
This button has a special function.
　このボタンには特別な機能があります。

【ビジネス英語】
The main function of this software is data analysis.
　このソフトウェアの主な機能はデータ分析です。

🎯 TOEICの出題頻度と傾向

  • 難易度：★★★☆☆
  • 出題パート：Part 5（文法・語彙問題）

TOEICではITやビジネス文脈で「機能」「役割」として頻出します。特に説明文や仕様書でよく見かけます。

❓ クイズに挑戦！

正しい使い方はどっち？

  1. The function of this device is to print documents.
  2. The function of this device is printing documents.

✅ クイズの答えと解説

  • 正解：1
  • 解説：function of ～ is to + 動詞の形が自然です。

🤔 role との違いは？

  • function：物や人が果たす「機能」や「働き」に重点
  • role：人や組織が担う「役割」や「立場」に重点

functionは「何をするか」という働き自体、roleは「どんな立場・責任か」に焦点があります。

  • purpose：目的
