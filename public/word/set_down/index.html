【set down】の意味・使い方・例文 ― put downとの違い【TOEIC英単語】

🔰 set downの発音と意味

[句動詞] 書き留める
[句動詞] 置く

「set down」は、物をそっと置く、または考えや情報を紙などに書き留めるときに使われます。特に公式な記録やメモを残す場面でよく使われます。

「set down」は古英語の「settan」（置く）と「down」（下へ）から成り立っています。動詞「set」に副詞「down」が加わることで、物理的に下に置く意味や、情報を記録する意味が生まれました。

✏️ set downの例文

【日常英会話】
Please set down your bag here.
　ここにあなたのかばんを置いてください。

She set down her thoughts in a notebook.
　彼女は自分の考えをノートに書き留めました。

【ビジネス英語】
The secretary set down the meeting minutes accurately for future reference.
　秘書は今後の参考のために会議の議事録を正確に書き留めました。

自然な使い方はどっち？

  1. The manager set down the new rules in the handbook.
  2. The manager set down the new rules on the table.

✅ クイズの答えと解説

  • 正解：1
  • 解説：1は「規則を書き留める」という意味で正しいです。2は「規則をテーブルの上に置く」となり、不自然です。

🤔 put down との違いは？

  • set down：落ち着いて記録したり、丁寧に置いたりする印象があります。
  • put down：単に物を下に置く動作を表し、記録の意味はありません。

「set down」は記録や書き留める意味でも使われますが、「put down」は単に物を置く動作に限られます。記録や文章化の場面では「set down」を使います。
