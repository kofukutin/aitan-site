【try out】の意味・使い方・例文 ― testとの違い【TOEIC英単語】

🔰 try outの発音と意味

[句動詞] 試してみる
[名詞] 選抜試験（可算）

try outは、何かを実際に使ったり行ったりして、その効果や使い心地を確かめるときに使います。また、スポーツや活動の選抜試験を指すこともあります。日常会話やビジネスの場面で、商品や方法などを試すときによく使われます。

try outは、古フランス語の“trier”（選ぶ、試す）に由来する“try”と、外に出す・実際に行うという意味の“out”が組み合わさった句動詞です。英語では16世紀ごろから使われ始め、何かを実際に試してみるという意味で定着しました。

✏️ try outの例文

【日常会話】
I want to try out this new game.
　この新しいゲームを試してみたいです。

She tried out for the soccer team.
　彼女はサッカーチームの選抜試験を受けました。

【ビジネス英語】
Before we launch the product, we should try out the new features to ensure there are no issues.
　製品を発売する前に、新機能を試して問題がないか確認すべきです。

🎯 TOEICの出題頻度と傾向

  • 難易度：★★☆☆☆
  • 出題パート：Part5（短文穴埋め問題）

Part5では句動詞の意味や使い方を問う問題で頻出します。try outは日常的な表現としてもよく出題されます。

正しい使い方はどっち？

  1. He try out the cake before buying it.
  2. He tried out the cake before buying it.

✅ クイズの答えと解説

  • 正解：2
  • 解説：2は過去の出来事なので時制が正しく、1は動詞の形が誤っています。

🤔 test との違いは？

  • try out：実際に使ったり体験したりして確かめる軽い感じです。
  • test：性能や品質を評価するために調べる、やや客観的な印象です。

try outは実際に使ってみる体験的なニュアンスが強く、testは性能や品質を評価するために調べる客観的な場面で使われやすいです。
