【run out】の意味・使い方・例文 ― use upとの違い【TOEIC英単語】

🔰 run outの発音と意味

[句動詞] 使い果たす
[句動詞] 尽きる

何かの物や資源がなくなってしまう、または使い切ってしまうことを表します。日常会話やビジネスの場面で、在庫や時間、エネルギーなどが尽きる場合によく使われます。

「run」は古英語の「rinnan（走る）」に由来し、「out」は外へを意味する前置詞です。組み合わせることで「外へ走り出る」から転じて「尽きる」「使い果たす」という意味になりました。

✏️ run outの例文

【日常英会話】
We ran out of milk this morning.
　今朝、牛乳がなくなりました。

My phone ran out of battery.
　私の携帯の電池が切れました。

【ビジネス英語】
During the meeting, we ran out of time before we could discuss the final topic.
　会議中に、最後の議題を話し合う前に時間がなくなってしまいました。

🎯 TOEICの出題頻度と傾向

  • 難易度：★★☆☆☆
  • 出題パート：Part3（会話問題）

会議やビジネスの会話文で、在庫や時間切れなどの文脈でよく出題されます。Part3では聞き取り問題で頻出です。

❓ クイズに挑戦！

自然な使い方はどっち？

  1. We ran out of paper, so we need to buy more.
  2. We ran out with paper, so we need to buy more.

✅ クイズの答えと解説

  • 正解：1
  • 解説：1は「紙がなくなった」という意味で正しい使い方です。2は「run out with」となっており、意味が通じません。

🤔 use up との違いは？

  • run out：何かが完全になくなってしまう、残りがゼロになる印象です。
  • use up：すべてを使い切ることに重点があり、計画的に消費しきるニュアンスがあります。

run outは自然に物がなくなる場合や予期せぬ不足に使われますが、use upは意図的にすべて使い切る場合に使われやすいです。
