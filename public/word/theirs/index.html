【theirs】の意味・使い方・例文 ― oursとの違い【TOEIC英単語】

🔰 theirsの発音と意味

[代名詞] 彼らのもの

「彼らのもの」「それらのもの」という意味で、前に出てきた名詞を繰り返さずに所有を表すときに使います。人や物の所有関係を簡潔に示すときによく使われます。

「theirs」は古英語の「their」（彼らの）に所有代名詞の語尾「-s」が付いた形です。英語の所有代名詞の一般的な構造に従っています。

✏️ theirsの例文

【日常会話】
This bag is theirs.
　このかばんは彼らのものです。

The books on the table are theirs.
　テーブルの上の本は彼らのものです。

【ビジネス英語】
If you are unsure which seats are theirs, please ask the receptionist for assistance.
　どの席が彼らのものか分からない場合は、受付係にお尋ねください。

自然な使い方はどっち？

  1. This umbrella is theirs.
  2. This umbrella is they.

🤔 ours との違いは？

  • theirs：所有関係を簡潔に表し、誰のものかを明確に示します。
  • ours：自分たちの所有を表し、話し手とそのグループに属するものを指します。

theirsは「彼らのもの」と第三者の所有を指し、oursは「私たちのもの」と話し手側の所有を指します。使う場面で所有者が異なります。
