【externally】の意味・使い方・例文 ― outwardlyとの違い【TOEIC英単語】

🔰 externallyの発音と意味

[副詞] 外部で
[副詞] 外面的に

物事が外側や外部で起こること、または外見や外部から見た様子を表す言葉です。主に物理的な外部や、システム・組織の外側を指す場合に使われます。

この単語はラテン語の “externus”（外側の）に由来し、英語の形容詞 “external”（外部の）に副詞を作る接尾辞 “-ly” が付いてできています。

✏️ externallyの例文

【日常会話】
The device can be powered externally.
　その装置は外部から電源を供給できます。

Externally, the building looks new.
　外見上、その建物は新しく見えます。

【ビジネス英語】
The software allows users to access data externally through a secure network connection.
　そのソフトウェアは、ユーザーが安全なネットワーク接続を通じて外部からデータにアクセスできるようにします。

🎯 TOEICの出題頻度と傾向

  • 難易度：★★★★☆
  • 出題パート：Part7（文書問題）

ITやビジネス文書の説明文で、システムや装置の外部接続や外部要因について述べる際によく出題されます。

❓ クイズに挑戦！

正しい使い方はどっち？

  1. The server can be accessed externally by authorized users.
  2. The server can be accessed internally by authorized users.

✅ クイズの答えと解説

  • 正解：1
  • 解説：1は外部からアクセスできるという意味で、externallyの正しい使い方です。2は内部からのアクセスを意味し、externallyの意味とは異なります。

【スタディサプリ ENGLISH】

🤔 outwardly との違いは？

  • externally：外側や外部に関する事柄を客観的に表します。
  • outwardly：表面上や見かけに関する印象をやや強調します。

externallyは物理的・機能的な外部を指すのに対し、outwardlyは見た目や表面的な印象に焦点を当てる点が異なります。
