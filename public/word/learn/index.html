【learn】の意味・使い方・例文 ― studyとの違い【TOEIC英単語】

🔰 learnの発音と意味

[動詞] 学ぶ、習得する、知る

learn は、「知識や技術を身につける」「経験から知る」といった意味で使われる動詞です。自分の努力や経験によって新しいことを理解したり覚えたりする場面で使います。

語源は古英語の leornian（学ぶ）に由来し、「知識を得る」という意味が長く使われてきました。

他の品詞：

  • learner：学習者（名詞）
  • learning：学習（名詞）

✏️ learnの例文

【日常会話】
I want to learn how to cook.
　料理の仕方を学びたいです。

【ビジネス英語】
We need to learn from our mistakes.
　私たちは失敗から学ぶ必要があります。

❓ クイズに挑戦！

正しい使い方はどっち？

  1. I want to learn English quickly.
  2. I want to learn English hardly.

✅ クイズの答えと解説

  • 正解：1
  • 解説：「quickly（速く）」はlearnと自然に使えますが、「hardly（ほとんど～ない）」は意味が合いません。

🤔 study との違いは？

  • learn：知識やスキルを身につける、結果として「できるようになる」ニュアンス
  • study：勉強する、知識を得るために努力する過程を強調

learnは「できるようになる」ことに重点があり、studyは「勉強する行為」自体を指します。

🧩 あわせて覚えたい

  • acquire：（努力して）習得する
  • teach：教える

📖 できればこれも

  • understand：理解する
  • practice：練習する
