【first name】の意味・使い方・例文 ― surnameとの違い【TOEIC英単語】

🔰 first nameの発音と意味

[名詞] 名（可算）

英語圏で使われる個人の名前のうち、名字ではなく本人固有の名前を指します。日本語の「名」にあたります。

「first name」は英語の「first（最初の）」と「name（名前）」が組み合わさった表現です。英語圏では名前が名字より先に書かれることからこの呼び方が定着しました。

✏️ first nameの例文

【日常会話】
My first name is John.
　私の名前はジョンです。

Please write your first name here.
　ここにあなたの名を書いてください。

【ビジネス英語】
During the interview, the manager asked me to spell my first name clearly for the record.
　面接中、記録のためにマネージャーが私の名をはっきりと綴るように求めました。

🎯 TOEICの出題頻度と傾向

  • 難易度：★★☆☆☆
  • 出題パート：Part3（会話問題）

Part3やPart7で、個人情報の確認やフォーム記入の場面でよく登場します。姓と名の区別を問う設問も見られます。

❓ クイズに挑戦！

正しい使い方はどっち？

  1. Please write your first name in this box.
  2. Please write your first name after your surname.

✅ クイズの答えと解説

  • 正解：1
  • 解説：1は「この欄に名を書いてください」と正しい使い方です。2は「姓の後に名を書いてください」となり、英語圏の一般的な順序と異なります。

🤔 surname との違いは？

  • first name：個人の名前そのものを指し、形式的でもカジュアルでも使えます。
  • surname：家族や一族を示す名字を指し、よりフォーマルな場面や書類で使われやすいです。

first nameは個人の名を指し、親しみやすさやカジュアルな場面でも使われますが、surnameは家族名を示し、よりフォーマルな書類や公式な場面で使われることが多いです。
