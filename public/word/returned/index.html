【returned】の意味・使い方・例文 ― refundedとの違い【TOEIC英単語】

🔰 returnedの発音と意味

[形容詞] 返却された、戻された

returned は、「返却された」「戻された」という意味で、主に商品や書類などが元の場所や持ち主に戻された状態を表します。

語源は「return（戻る、返す）」の過去分詞形で、「再び（re-）回る（turn）」から「元に戻す」という意味が派生しました。

他の品詞：

  • return：戻る、返す（動詞）
  • return：返却、帰還（名詞）

✏️ returnedの例文

【日常会話】
The library book is returned.
　図書館の本は返却されています。

【ビジネス英語】
All returned items must be inspected.
　すべての返却品は検査されなければなりません。

🎯 TOEICの出題頻度と傾向

  • 難易度：★★☆☆☆

TOEICでは、受動態や過去分詞の形容詞的用法として「returned item（返却品）」などの形でよく出題されます。返品や返却に関するビジネス文脈で頻出です。

❓ クイズに挑戦！

正しい使い方はどっち？

  1. The returned package was damaged.
  2. The return package was damaged.

✅ クイズの答えと解説

  • 正解：1
  • 解説：「returned」は「返却された」という意味の形容詞で、返送済みの荷物を表します。「return package」だと「返送用の荷物」となり意味が異なります。

🤔 refunded との違いは？

  • returned：返却された、元に戻された状態を表す
  • refunded：返金された、お金が戻された状態を表す

「returned」は物や書類が元に戻ること、「refunded」は支払ったお金が戻ることを指します。
