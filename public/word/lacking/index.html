【lacking】の意味・使い方・例文 ― missingとの違い【TOEIC英単語】

🔰 lackingの発音と意味

[形容詞] 不足している

必要なものや期待されるものが十分にない、または欠けている状態を表します。主にbe動詞とともに使われ、何かが足りないことを説明するときに用いられます。

この語は動詞lack（不足する、欠けている）に形容詞化の接尾辞-ingが付いた形です。語源は古英語のlac（欠如、損失）に由来しています。

✏️ lackingの例文

【日常会話】
Her explanation was lacking in detail.
　彼女の説明は詳細が不足していました。

The soup is lacking flavor.
　そのスープは味が足りません。

【ビジネス英語】
The report was lacking sufficient data to support its conclusions, which raised concerns among the managers.
　その報告書は結論を裏付ける十分なデータが不足していたため、マネージャーたちの間で懸念が生じました。

🎯 TOEICの出題頻度と傾向

  • 難易度：★★★☆☆
  • 出題パート：Part5（短文穴埋め問題）

Part5の文法問題やPart7の長文読解で、不足や欠如を表す表現として頻出します。特にbe動詞とセットで使われることが多いです。

❓ クイズに挑戦！

自然な使い方はどっち？

  1. The report is lacking important details, so we need to revise it.
  2. The report is lacking with important details, so we need to revise it.

✅ クイズの答えと解説

  • 正解：1
  • 解説：正解は1です。「lacking」は目的語を直接取るため、前置詞withは不要です。2は文法的に誤りです。

🤔 missing との違いは？

  • lacking：必要なものが十分にない、または欠けている状態を淡々と述べる印象です。
  • missing：missingは「本来あるべきものがなくなっている」や「行方不明」というニュアンスが強いです。

lackingは「十分でない」「不足している」と全体的な量や質の不足を表し、missingは「あるべきものが存在しない」や「紛失している」ことを強調します。
