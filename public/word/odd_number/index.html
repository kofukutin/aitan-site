🔰 odd numberの発音と意味

[名詞] 奇数（可算）

2で割り切れない整数のことを指します。たとえば1、3、5などが該当します。数学や算数の授業でよく使われます。

oddは古ノルド語の“oddi”に由来し、「端」や「余り」を意味します。numberはラテン語の“numerus”から来ており、「数」を表します。2語の組み合わせで「奇数」という意味になります。

✏️ odd numberの例文

【日常会話】
Seven is an odd number.
　7は奇数です。

Is nine an odd number?
　9は奇数ですか？

【ビジネス英語】
In this math test, please circle all the odd numbers you can find in the list below.
　この数学のテストでは、下のリストから見つけられるすべての奇数に丸をつけてください。

❓ クイズに挑戦！

正しい使い方はどっち？

  1. Twelve is an odd number.
  2. Fifteen is an odd number.

  • 正解：2

🤔 even number との違いは？

  • odd number：数学的に2で割り切れない数を指し、特別な感情や評価は含みません。
  • even number：2で割り切れる数を指し、偶数として区別されます。

odd numberは2で割り切れない数、even numberは2で割り切れる数を表します。数学や算数の説明で使い分けます。
