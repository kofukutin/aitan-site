【ours】の意味・使い方・例文 ― ourselvesとの違い【TOEIC英単語】

🔰 oursの発音と意味

[代名詞] 私たちのもの

「私たちのもの」という意味で、所有を表すときに使います。すでに話題に出ている名詞を繰り返さずに指すときに便利です。

この単語は古英語の「ūser」から派生し、「our（私たちの）」に所有代名詞の形を示す-sが付いた形です。英語の所有代名詞の一つとして体系的に使われています。

✏️ oursの例文

【日常英会話】
This bag is ours.
　このかばんは私たちのものです。

Is this seat ours?
　この席は私たちのものですか？

【ビジネス英語】
If the proposal is accepted, the responsibility will be entirely ours to manage the project efficiently.
　提案が受け入れられた場合、プロジェクトを効率的に管理する責任は完全に私たちのものとなります。

🎯 TOEICの出題頻度と傾向

  • 難易度：★★☆☆☆
  • 出題パート：Part5（短文穴埋め問題）

Part5の文法問題やPart7の長文読解で、所有代名詞の使い分けを問う問題が出題されることがあります。

❓ クイズに挑戦！

自然な使い方はどっち？

  1. This umbrella is ours, so please don’t take it.
  2. This umbrella is ours’ so please don’t take it.

✅ クイズの答えと解説

  • 正解：1
  • 解説：正解はoursで、所有代名詞にアポストロフィは不要です。不正解は文法的に誤りです。

🤔 ourselves との違いは？

  • ours：所有物や権利が自分たちに属していることをはっきり示します。
  • ourselves：自分たち自身を強調したいときに使われ、所有ではなく主体や行為者を指します。

oursは「私たちのもの」と所有を表し、ourselvesは「私たち自身」と主体や強調を表すため、使う場面や意味が異なります。
