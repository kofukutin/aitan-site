【logging】の意味・使い方・例文 ― recordとの違い【TOEIC英単語】

🔰 loggingの発音と意味

[名詞] 記録（不可算）、伐採（不可算）

logging は、IT分野では「記録すること」、特にシステムやアプリケーションの動作記録を指します。また、一般的には「森林伐採」という意味もあります。

語源は「log（丸太、記録）」＋「-ing（動作）」で、もともとは木を切る行為から、記録する行為へと意味が広がりました。

他の品詞：

  • log：記録する、丸太（動詞・名詞）
  • logger：記録装置、伐採作業員（名詞）

✏️ loggingの例文

【日常会話】
The computer is logging all the changes.
　コンピューターがすべての変更を記録しています。

【ビジネス英語】
Proper logging helps us track system errors efficiently.
　適切なログ記録はシステムエラーの効率的な追跡に役立ちます。

🎯 TOEICの出題頻度と傾向

  • 難易度：★★★☆☆
  • 出題パート：Part 7（長文読解・説明文）

ITやビジネス関連の長文で、システムの記録や管理に関する文脈で出題されることが多い単語です。

正しい使い方はどっち？

  1. The software supports automatic logging of user activity.
  2. The software supports automatic log of user activity.

✅ クイズの答えと解説

  • 正解：1
  • 解説：loggingは「記録すること」という動作名詞で、ここではactivityの記録という意味に自然です。

🤔 record との違いは？

  • logging：システムや機械的な「記録する行為」や「記録そのもの」を指すことが多い
  • record：広く「記録」「記録すること」全般を指し、紙や音声など多様な媒体に使える

loggingはITや技術分野での自動的・継続的な記録に使われ、recordはより一般的な「記録」や「記録する」という意味で使われます。

🧩 あわせて覚えたい

  • monitor：監視する、モニターする
  • audit：監査する、検査する

📖 できればこれも

  • database：データベース、情報の集まり
  • track：追跡する、記録する
