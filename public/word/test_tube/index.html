【test tube】の意味・使い方・例文 ― beakerとの違い【TOEIC英単語】

🔰 test tubeの発音と意味

[名詞] 試験管（可算）

液体や化学物質を少量入れて実験や観察を行うための細長いガラス製の容器を指します。主に理科や化学の実験で使われます。

「test」はラテン語の「testa（陶器の壺）」に由来し、英語で「試験」や「実験」を意味します。「tube」はラテン語の「tubus（管）」が語源で、英語で「管」や「筒」を表します。これらが組み合わさって「test tube」となり、実験用の管という意味になりました。

✏️ test tubeの例文

【日常英会話】
Please put the liquid into the test tube.
　その液体を試験管に入れてください。

The test tube broke when it fell.
　試験管は落ちたときに割れました。

【ビジネス英語】
During the experiment, the scientist carefully heated the solution in the test tube to observe the reaction.
　実験中、科学者は反応を観察するために試験管内の溶液を慎重に加熱しました。

自然な使い方はどっち？

  1. The student mixed the chemicals in a test tube.
  2. The student mixed the chemicals on a test tube.

✅ クイズの答えと解説

  • 正解：1
  • 解説：1は「試験管の中で混ぜる」という正しい使い方です。2は「試験管の上で混ぜる」となり、不自然です。

🤔 beaker との違いは？

  • test tube：理科や化学の実験で使う細長いガラス容器という印象です。
  • beaker：広口で液体を注いだり混ぜたりするための容器という印象です。

test tubeは細長い形状で少量の液体を扱う実験用容器を指しますが、beakerは広口で容量が大きく、液体を注いだり混ぜたりする用途に使われます。
