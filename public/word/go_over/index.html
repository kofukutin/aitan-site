【go over】の意味・使い方・例文 ― reviewとの違い【TOEIC英単語】

🔰 go overの発音と意味

[句動詞] 詳しく調べる
[句動詞] 繰り返し確認する

何かを一つ一つ丁寧に確認したり、内容を見直したりする時に使います。会議や授業などで、資料や内容を順にチェックする場面でよく使われます。

「go」は古英語の“gān”に由来し、「over」は古英語の“ofer”から来ています。句動詞「go over」は「go（行く）」と「over（越えて）」が組み合わさり、「上から下まで確認する」「全体を見直す」という意味になりました。

✏️ go overの例文

【日常英会話】
Let’s go over the answers together.
　一緒に答えを確認しましょう。

Can you go over this report before the meeting?
　会議の前にこの報告書を確認してもらえますか？

【ビジネス英語】
Before we finalize the contract, let’s go over all the terms to make sure everything is correct.
　契約を最終決定する前に、すべての条件を確認して間違いがないか見ておきましょう。

🎯 TOEICの出題頻度と傾向

  • 難易度：★★☆☆☆
  • 出題パート：Part5（短文穴埋め問題）

Part5では文脈に合った句動詞の選択肢として出題されることが多いです。会議や資料確認の場面で頻出です。

❓ クイズに挑戦！

自然な使い方はどっち？

  1. We need to go over the details before making a decision.
  2. We need to go over to the store before making a decision.

✅ クイズの答えと解説

  • 正解：1
  • 解説：1は「詳細を確認する」という意味で正しいです。2は「店に行く」という意味になり、文脈に合いません。

🤔 review との違いは？

  • go over：一つ一つ丁寧に確認する落ち着いた印象です。
  • review：全体を見直すというより、内容を再度学習するニュアンスが強いです。

go overは内容を順に確認する場面で使われ、reviewは復習や再学習の場面で使われやすいです。
