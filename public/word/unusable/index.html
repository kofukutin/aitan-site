【unusable】の意味・使い方・例文 ― brokenとの違い【TOEIC英単語】

🔰 unusableの発音と意味

[形容詞] 使用できない

何かが壊れていたり、損傷していたり、条件が悪いために使うことができない状態を表します。物やシステム、道具などが本来の目的で利用できない場合によく使われます。

この単語は接頭辞un-（否定）と形容詞usable（使える）が組み合わさってできています。usableはラテン語の「uti（使う）」に由来し、un-を付けることで「使えない」という意味になります。

✏️ unusableの例文

【日常英会話】
This pen is unusable because it has no ink.
　このペンはインクがないので使えません。

The old computer became unusable after the update.
　その古いコンピューターはアップデート後に使えなくなりました。

【ビジネス英語】
Due to a critical software error, the application was rendered unusable for all employees until the issue was fixed.
　重大なソフトウェアエラーのため、そのアプリケーションは問題が解決されるまで全従業員が使えない状態になりました。

🎯 TOEICの出題頻度と傾向

  • 難易度：★★★☆☆
  • 出題パート：Part5（短文穴埋め問題）

Part5の文法問題やPart7の長文で、機器やシステムの状態説明として出題されることがあります。否定的な意味合いに注意が必要です。

❓ クイズに挑戦！

自然な使い方はどっち？

  1. The file was unusable after it got corrupted.
  2. The file was unusable before it got corrupted.

✅ クイズの答えと解説

  • 正解：1
  • 解説：1はファイルが壊れた後に使えなくなったという意味で正しいです。2は壊れる前から使えなかったという意味になり、文脈に合いません。

🤔 broken との違いは？

  • unusable：完全に使えない、役に立たないという否定的な印象があります。
  • broken：壊れている状態を強調し、物理的な損傷や故障を直接示します。

unusableは「使えない状態」全般を指し、必ずしも物理的に壊れているとは限りませんが、brokenは主に物理的な損傷や故障を強調します。

  • obsolete（時代遅れの）
