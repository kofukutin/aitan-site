【incompatible】の意味・使い方・例文 ― contradictoryとの違い【TOEIC英単語】

🔰 incompatibleの発音と意味

[形容詞] 両立できない
[形容詞] 互換性がない

二つ以上のものが一緒に存在したり、同時に使われたりすることができない状態を表します。特にソフトウェアや機器などが互いに適合しない場合によく使われます。

この単語はラテン語の “in-"（否定）と “compati”（共に耐える）が語源です。接頭辞 “in-” が「〜でない」、語幹 “compatible”（互換性がある）により、「互換性がない」という意味になりました。

✏️ incompatibleの例文

【日常英会話】
This app is incompatible with my phone.
　このアプリは私の携帯電話と互換性がありません。

Some foods are incompatible with certain medicines.
　いくつかの食品は特定の薬と一緒に摂取できません。

【ビジネス英語】
The new software update is incompatible with older operating systems, causing issues for many users.
　新しいソフトウェアのアップデートは古いオペレーティングシステムと互換性がなく、多くのユーザーに問題を引き起こしています。

🎯 TOEICの出題頻度と傾向

  • 難易度：★★★★☆
  • 出題パート：Part7（文書問題）

Part7の長文読解やIT関連の説明文で、システムや製品の互換性について述べる際によく出題されます。

❓ クイズに挑戦！

自然な使い方はどっち？

  1. These two software programs are incompatible, so they cannot run together on the same computer.
  2. These two software programs are incompatible, so they always work perfectly together.

✅ クイズの答えと解説

  • 正解：1
  • 解説：incompatibleは「両立できない」という意味です。2番は「常にうまく動作する」となっており、語義に反します。

🤔 contradictory との違いは？

  • incompatible：一緒に使うことができない、または共存できないという否定的な印象があります。
  • contradictory：互いに矛盾していて、同時に成立しないという強い対立のニュアンスがあります。

incompatibleは主に物やシステムが一緒に使えない場合に使われ、contradictoryは意見や主張などが真っ向から対立している場合に使われます。
