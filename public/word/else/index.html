【else】の意味・使い方・例文 ― anotherとの違い【TOEIC英単語】

🔰 elseの発音と意味

[副詞] 他に、ほかに

「他に」や「ほかに」という意味で、何か追加や別のものを指すときに使います。主に疑問詞やsome、anyなどと一緒に使われることが多いです。

この単語は古英語の“elles”に由来し、「他の方法で」や「他に」という意味を持っていました。単独の語幹で構成されており、接頭辞や接尾辞はありません。

✏️ elseの例文

【日常会話】
Do you want anything else?
　他に何か欲しいですか？

Is there anyone else in the room?
　部屋に他に誰かいますか？

【ビジネス英語】
If you have any questions about the project, please let me know if there is anything else I can help with.
　プロジェクトについてご質問があれば、他にお手伝いできることがあればお知らせください。

🎯 TOEICの出題頻度と傾向

  • 難易度：★★☆☆☆
  • 出題パート：Part5（短文穴埋め問題）

Part5などの文法問題で、疑問詞やsome、anyと一緒に使われる形でよく出題されます。意味や語順に注意が必要です。

❓ クイズに挑戦！

自然な使い方はどっち？

  1. If you want something else, please let me know.
  2. If you want something else, please let me about.

✅ クイズの答えと解説

  • 正解：1
  • 解説：正解は1です。’let me know’が正しい表現で、2の’let me about’は不自然な使い方です。

🤔 another との違いは？

  • else：他に何か、他の人や物を指すときに幅広く使われます。
  • another：別の一つを強調したいときに使われます。数量が一つであることが明確な場合に使われることが多いです。

elseは「他に」と幅広く使えますが、anotherは「もう一つの」「別の一つの」と数量が一つであることを強調したいときに使います。

🧩 あわせて覚えたい

  • other（他の）
  • besides（そのほかに）
