【photoreactive】の意味・使い方・例文 ― photosensitiveとの違い【TOEIC英単語】

🔰 photoreactiveの発音と意味

[形容詞] 光に反応して化学変化を起こす

光を受けることで化学反応を起こす性質を持つことを表します。主に科学や技術分野で、光によって活性化される物質や材料について使われます。

photoreactiveは、ギリシャ語の「phōs（光）」を意味する「photo-」という接頭辞と、ラテン語の「reactivus（反応する）」に由来する「reactive（反応性の）」が組み合わさってできた語です。光に対して反応する性質を表現しています。

✏️ photoreactiveの例文

【日常英会話】
This material is photoreactive.
　この素材は光に反応します。

Some chemicals are photoreactive under sunlight.
　いくつかの化学物質は日光の下で光反応性を示します。

【ビジネス英語】
The company developed a photoreactive coating that changes color when exposed to ultraviolet light.
　その会社は紫外線にさらされると色が変わる光反応性コーティングを開発しました。

❓ クイズに挑戦！

正しい使い方はどっち？

  1. The medicine is photoreactive, so it should be kept in the dark.
  2. The paint is photoreactive, so it changes color when exposed to light.

✅ クイズの答えと解説

  • 正解：2
  • 解説：2は光に当たると色が変わるという光反応性の正しい使い方です。1は薬を暗所に保管する理由としては不自然で、文脈が適切ではありません。

🤔 photosensitive との違いは？

  • photoreactive：光によって化学的な変化や反応を起こす性質を強調します。
  • photosensitive：光に対して敏感であることを表し、必ずしも化学反応を伴うとは限りません。

photoreactiveは光によって実際に化学反応や変化が起こる場合に使い、photosensitiveは光に対して感度が高い場合や、光で影響を受けやすい場合に使います。
