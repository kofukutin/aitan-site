【incompletely】の意味・使い方・例文 ― partiallyとの違い【TOEIC英単語】

🔰 incompletelyの発音と意味

[副詞] 不完全に

物事が完全に行われていない状態や、全体の一部しか達成されていない様子を表します。主に書き言葉やフォーマルな場面で使われます。

この単語は接頭辞in-（否定）と形容詞complete（完全な）、そして副詞を作る接尾辞-lyから成り立っています。語源はラテン語のcompletus（満たされた）に由来します。

✏️ incompletelyの例文

【日常会話】
The project was incompletely finished.
　そのプロジェクトは不完全に終わりました。

He answered the question incompletely.
　彼はその質問に不完全に答えました。

【ビジネス英語】
The report was submitted incompletely, missing several important sections required by the manager.
　その報告書は不完全な状態で提出され、マネージャーが求めていた重要な部分がいくつか抜けていました。

🎯 TOEICの出題頻度と傾向

  • 難易度：★★★★☆
  • 出題パート：Part7（文書問題）

Part7の長文読解や説明文で、不完全な状態や部分的な達成を表す際によく見かけます。選択肢や本文中で副詞として使われることが多いです。

❓ クイズに挑戦！

自然な使い方はどっち？

  1. The project was incompletely finished, so we need to do more work.
  2. The project was incompletely started, so it is already done.

✅ クイズの答えと解説

  • 正解：1
  • 解説：正解は1です。「incompletely」は「不完全に」という意味なので、1は文脈に合っています。2は「始めたのにすでに終わっている」と文脈が不自然です。

🤔 partially との違いは？

  • incompletely：全体が満たされていない、または一部が欠けている印象を与えます。
  • partially：全体の一部だけを扱う、または限定的な範囲であることを強調します。

incompletelyは『完全でない』ことを強調し、全体の不足や欠落を示します。一方、partiallyは『部分的に』という意味で、全体の一部だけが対象であることを示しますが、必ずしも不足や欠落を強調しません。
