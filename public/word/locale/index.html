【locale】の意味・使い方・例文 ― locationとの違い【TOEIC英単語】

🔰 localeの発音と意味

[名詞] 場所、現場、（特定の）地域

locale は、特定の出来事や活動が起こる場所や地域、またはIT分野では言語や地域設定を指します。

語源はラテン語の「locus（場所）」から派生し、フランス語経由で英語に入ったため、ややフォーマルな響きを持ちます。

他の品詞：

  • local：地元の、地域の（形容詞）
  • locality：場所、近所（名詞）

✏️ localeの例文

【日常会話】
This cafe is a popular locale for students.
　このカフェは学生たちに人気の場所です。

【ビジネス英語】
Please set the locale to Japanese in the application settings.
　アプリの設定でロケールを日本語にしてください。

🎯 TOEICの出題頻度と傾向

  • 難易度：★★★★☆
  • 出題パート：Part 7（長文読解）

localeはITやビジネス文書、説明文などで見かけることが多く、TOEICでは主にPart 7の長文やマニュアル系の問題で出題されます。

🤔 location との違いは？

  • locale：特定の出来事や活動が行われる場所、またはITでの地域設定
  • location：単に「場所」「位置」を指す一般的な語

localeは「雰囲気」や「文化的背景」も含む場合や、IT分野での「言語・地域設定」を指す時に使われますが、locationは物理的な位置や住所を表すのが主です。
