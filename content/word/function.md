[名詞] 機能、役割、関数（可算）

**function** は、「物や人が果たす役割」や「機能」、またITや数学分野では「関数」という意味で使われます。

語源はラテン語の「fungi（果たす）」から来ており、「何かを果たすこと」が基本イメージです。

他の品詞：  
- functional：機能的な（形容詞）
- functionally：機能的に（副詞）

---

## ✏️ functionの例文

【日常会話】  
This button has a special function.  
　このボタンには特別な機能があります。

【ビジネス英語】  
The main function of this software is data analysis.  
　このソフトウェアの主な機能はデータ分析です。

## 🤔  role との違いは？

- **function**：物や人が果たす「機能」や「働き」に重点
- **[role](/role)**：人や組織が担う「役割」や「立場」に重点

functionは「何をするか」という働き自体、roleは「どんな立場・責任か」に焦点があります。
