[名詞] 概要、全体像（可算）

**overview** は、物事の全体像や大まかな説明を示すときに使われる名詞です。主にレポートやプレゼン、説明文などで「全体の流れ」や「大枠」を伝える際に使われます。

語源は「over（上から）」＋「view（見る）」で、「上から全体を見渡す」というイメージから来ています。

他の品詞：  
- overviewed：ざっと見渡した（動詞・過去形）
- overviewing：ざっと見渡している（動詞・現在分詞）

## ✏️ overviewの例文

【日常会話】  
Can you give me an overview of the movie?  
　その映画の概要を教えてくれる？

【ビジネス英語】  
The manager provided an overview of the new project.  
　マネージャーが新しいプロジェクトの概要を説明しました。

## 🎯 TOEICの出題頻度と傾向

- 難易度：★★☆☆☆
- 出題パート：Part 7（長文読解・説明文）

TOEICでは主にPart 7の説明文やメール、レポートの冒頭で「概要」や「全体像」を示す語としてよく登場します。ビジネス文書で頻出です。

正しい使い方はどっち？

1. Please read the overview before starting the test.  
2. Please read the overview before start the test.

---

## ✅ クイズの答えと解説

- 正解：**1**
- 解説：overviewの後は「before + 動名詞（starting）」が正しい形です。2は文法的に誤りです。

## 🤔  summary との違いは？

- **overview**：全体像や大まかな流れをざっくり説明する
- **[summary](/summary)**：要点や重要な部分を簡潔にまとめる

overviewは「全体を見渡す」イメージ、summaryは「要点を抜き出してまとめる」イメージで使い分けます。
